## HNG Task Stage One

## Requirements

* [Go](https://golang.org) -  v1.16 above

## Installation

* Clone this repo

  ```bash
  git clone https://github.com/olad5/hng-9-task-one.git
  ```

* Change directory to project directory

  ```bash
  cd hng-9-task-one
  ```

* Copy .env

  ```bash
  cp .env.example .env
  ```

* Run application

  ```bash
  go run main.go
  ```

  This will serve this application on the port :4000
